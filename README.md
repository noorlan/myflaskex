# Flaskex App

This is testing Flaskex App

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/)

```bash
pip install -r requirements.txt
```

## Setup

```python
git clone https://github.com/ikambarov/Flaskex.git
cd Flaskex
pip install -r requirements.txt
python app.py
```
